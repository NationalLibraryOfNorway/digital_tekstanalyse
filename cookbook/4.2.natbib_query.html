2.3. Søk Nasjonalbibliografiens metadata med dhlab#

from dhlab.metadata.natbib import metadata_query, metadata_from_urn, pretty_print_marc21json

q = [
        ["245", "a", "kongen"]
        ]

metadata_query(q, limit = 3)

['URN:NBN:no-nb_digibok_2010100609054',
 'URN:NBN:no-nb_digibok_2011061408114',
 'URN:NBN:no-nb_digibok_2017010348086']

metadata_query tar en liste med tupler, der hver tuppel en MARC 21 delfelt

urns = metadata_query(
    [
        ("100", "a", "Ibsen")
        ],
    limit=2
                    )
urns

['URN:NBN:no-nb_digibok_2007041101079', 'URN:NBN:no-nb_digibok_2006110601044']

metadata_from_urn tar en liste med URNer, og returnerer MARC 21 metadata i json-format

metadata = metadata_from_urn(urns, fields=["245", "100", "008"])
